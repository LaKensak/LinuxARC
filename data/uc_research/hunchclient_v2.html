Requirements

ARC Raiders running on Linux (Proton/Wine)

Python 3

GTK4 + PyGObject

Root access (reads /proc/PID/mem) (or just use the included kernel)

Installation

Arch:
Code:
sudo pacman -S --needed python python-gobject gtk4 cairo
Debian/Ubuntu:
Code:
sudo apt update && sudo apt install -y python3 python3-gi python3-gi-cairo gir1.2-gtk-3.0
Usage

Start the game, then run:
Code:
./start.sh
