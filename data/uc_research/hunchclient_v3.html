[Linux]hunchclient v3 external
Edit: OUTDATED DUE TO GAME UPDATE



my new and prob for now last vers of my arc raiders cheat, was really fun using this to learn, but i hit some roadblocks and it takes a lil bit too much time for my likings.
if anyone wanna help me improve it im down to update it if i can get more stuff working.

this version has a better version of the ESP i did, its better in performance, better looking (i guess) and all in all has more features.

includes two kernels this time, the mouse kernel is always running, since it is required for the aimbot. i included the src of all the kernels, and no compiled one. the install script should do that for u.

added in this version:
  • aimbot
  • aimbone selection (head, chest, pelvis)
  • easier filtering for wanted loot (only show mapped containers)
  • container name mapper
  • player dummy spawner (for aimbot testing)

features that are planned but dont work yet:
  • arc aimbot / arc weakpoint esp
  • arc health
  • container and item pickup names
  • gun ballistics

Requirements
  • ARC Raiders running on Linux (Proton/Wine)
  • Python 3
  • GTK4 + PyGObject + gtk4-layer-shell
  • Root access (or just use the included kernel)
  • Wayland (tested on Hyprland)

Installation

fresh system? just run this bad boy:
Code:
./install.sh
it detects ur package manager (pacman/apt/dnf), installs all deps, builds the kernels, basically does everything for u :3

Manual install if u want:

Arch:
Code:
sudo pacman -S --needed python python-pip python-gobject gtk4 gtk4-layer-shell base-devel linux-headers clang llvm cairo pango
Debian/Ubuntu:
Code:
sudo apt install -y python3 python3-pip python3-gi gir1.2-gtk-4.0 build-essential linux-headers-$(uname -r) clang llvm
then build kernels:
Code:
cd kernel && make
Usage

start the game, then:
Code:
./start.sh
pick ur mode (stealth kernel, normal kernel, or /proc/mem) and ur good to go

config ui at http://localhost:8080 while running

enjoy :3

info about safety ( all modes are tested and safe as of now (19.01.2026)

fastest will be the proc mem, maybe ill figure out shared mem anytime xd
aimbot is always use at your own risk, they do serverside detection and i dont do anytihng to fight that really, if u use it smart u should be fine but be warned
